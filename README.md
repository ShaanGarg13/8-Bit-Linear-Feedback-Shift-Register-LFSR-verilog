# 8-Bit-Linear-Feedback-Shift-Register-LFSR-verilog
Implementation of an 8-Bit Linear Feedback Shift Register (LFSR) in Verilog HDL with configurable seed loading, programmable feedback polynomial, pseudorandom sequence generation, RTL design, simulation, waveform analysis, and logic synthesis using Quartus II and ModelSim-Altera.
